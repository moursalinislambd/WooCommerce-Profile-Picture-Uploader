📸 **WooCommerce Profile Picture Uploader**
-------------------------------------------

**Let your customers upload profile pictures during registration!** A lightweight solution that adds photo upload functionality to WooCommerce with full admin control.

### ✨ **Features:**

*   🖼️ **Registration Upload** - Customers can add photos when signing up
    
*   👤 **Account Management** - View & change pictures in "My Account"
    
*   🎯 **Admin Control** - Full management from WordPress admin panel
    
*   🗑️ **Quick Delete** - Remove photos directly from users list
    
*   🛡️ **Smart Validation** - 2MB max size, supports JPG/PNG/GIF
    
*   🔒 **Secure** - Built-in WordPress security standards
    

### 📥 **Installation:**

**Method 1: Copy-Paste (2 minutes)**

1.  Go to **Appearance → Theme File Editor**
    
2.  Open **functions.php** of your child/active theme
    
3.  Copy the entire code and paste at the bottom
    
4.  Click **Update File**
    

**Method 2: Code Snippets Plugin (Recommended)**

1.  Install **"Code Snippets"** plugin
    
2.  Go to **Snippets → Add New**
    
3.  Title: WooCommerce Profile Picture
    
4.  Paste the code

5.  Click **Save Changes and Activate**
    

### 🎯 **That's it!** The feature will appear automatically on:

*   Registration page 📝
    
*   My Account → Account Details 👤
    
*   Admin Users panel ⚙️
